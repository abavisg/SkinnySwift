# SkinnySwift
##### Version 0.3.0
Helpful swift classes to speed up development

## Installation
### Cocoapods

SkinnySwift is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "SkinnySwift"
```
